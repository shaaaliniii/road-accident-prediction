## 📌 Overview

This project is a **Machine Learning-based Road Accident Prediction and Analysis System** designed to analyze accident data and predict the likelihood of accidents based on various factors.

It helps in understanding accident patterns and provides insights that can assist authorities and users in improving road safety.

---

## 🎯 Key Features

* 🚦 Accident prediction using trained ML model
* 📊 Data analysis and visualization
* 🗺️ Interactive accident map
* 📈 Insights from historical accident data
* 🌐 Web interface using Flask

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/shaaaliniii/road-accident-prediction.git
cd road-accident-prediction
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python main.py
```

Then open:
👉 http://127.0.0.1:5000
